Code of Conduct

RailsTokyoは、Ruby on Railsとその周辺技術をテーマに、学び合い・つながり合い・支え合うコミュニティイベントです。
私たちは、参加するすべての人が安心して交流し、自由に学び、前向きに挑戦できる場であることを大切にしています。
そのため、RailsTokyoでは以下の行動規範を定めています。

1. リスペクトと思いやりを忘れずに

すべての参加者・登壇者・スタッフは、互いの背景・経験・考えを尊重し合いましょう。
知識やスキルの違いを優劣ではなく「多様性」として受け入れ、互いに敬意を持って関わることを大切にします。

2. 安心・安全なコミュニケーションを

RailsTokyoは、誰もが安心して発言できる場を目指しています。
差別的・攻撃的・ハラスメントと感じられるような言動はお控えください(アンチハラスメントポリシーはこちら)。
相手の立場に立った、思いやりあるコミュニケーションを心がけましょう。

3. チャレンジを歓迎します

登壇も参加も、はじめの一歩を応援しています。
「まだ初心者だから…」という遠慮は必要ありません。
聞く人も、話す人も、互いのチャレンジを温かく受け止めましょう。

4. つながりは、未来への投資

RailsTokyoでの出会いや会話は、きっと次の学びやチャンスにつながります。
その場限りではない、継続的な関係を育てられるような、誠実でオープンな関わりを大切にしましょう。

私たちは、この行動規範がすべての参加者にとってポジティブな体験を生み出す土台になると信じています。
ルールというより「みんなで良い場をつくるための約束ごと」として、ぜひご理解とご協力をお願いいたします。
